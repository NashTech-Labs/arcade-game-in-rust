# Arcade Game using Rust Programming
This repository contains a source code of terminal based arcade game.

This game was initially developed for a presentation at [OSCON Open Source Software Superstream Series: Live Codingâ€”Go, Rust, and Python](https://learning.oreilly.com/live-training/courses/oscon-open-source-software-superstream-series-live-codinggo-rust-and-python/0636920410188/) and then adapted for inclusion as an example project for the [Ultimate Rust Crash Course](https://www.udemy.com/course/ultimate-rust-crash-course/?referralCode=AF30FAD8C6CCCC2C94F0).

This game contains sounds as well, so to get the sounds you can get from here [Sounds](https://github.com/CleanCut/invaders/files/6312508/sounds.zip).

## Modules Overview

### Player
This module contains all the activities and operations performed by the player.

### Invaders
This module contains the activities and operation related to enemies and here we are calling them Invaders.

### Shot
This contains the activities for a shot like(explode, generate, update).

### Render
Through this we are actually creating our play area.

### Frame
This helps us to provide a playing frame in our play area.
